# AI Automation Portfolio
Built by (Divyansh Sharma)[https://www.linkedin.com/in/divyansh-sharma-b4793026b/] — focused on AI automation, workflow design, and practical product integrations.
A collection of AI-powered automation projects built using **n8n**, AI tools, APIs, and workflow integrations.

This repository showcases practical automation systems designed to solve real-world business and productivity problems, including recruitment support, email assistance, content generation, lead processing, and web-based applications.

## Projects

| Project | Description | Status |

| **AI HR Recruitment Assistant** | Automates parts of the hiring workflow such as candidate intake, screening, analysis, and recruiter support. | In Progress |
| **AI Powered Email Assistant** | Helps automate email-related tasks such as drafting, summarization, classification, response support, or routing. | In Progress |

- `README.md` — project overview, workflow explanation, and setup notes
- `workflow.json` — exported n8n workflow file, where applicable
- `architecture.png` or screenshots — workflow architecture and visual reference
- Additional assets or notes if needed
